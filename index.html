6 editeurs de texte gratuits

BRACKET

Brackets est un éditeur de texte gratuit conçu principalement pour les concepteurs Web, mais il est bien entendu possible à quiconque de visualiser ou de modifier un document texte. L’interface est extrêmement clean et moderne et se révèle être vraiment facile à utiliser malgré tous ses paramètres avancés. En fait, presque toutes les options sont cachées de l’interface standard, permettant à quiconque d’utiliser cette application, qui offre également une interface utilisateur extrêmement ouverte pour la modification.

VIRTUAL STUDIO

Visual Studio Code est un éditeur de texte puissant et gratuit que vous aimeriez utiliser. Son design et sa puissance peuvent vous faire penser qu’il s’agit d’un environnement de développement ultra-complet. L’éditeur de texte pour Windows est capable de compléter le code contextuel car la technologie Intellisense y est intégrée. VS Code dispose d’un grand système d’extensions pouvant être facilement manipulé depuis le terminal fourni par VS Code. Vous pouvez télécharger et ajouter de nombreuses fonctionnalités intéressantes, telles que serveur en direct, Le support pour Python, des codes Angular 6, un gestionnaire de fichiers, etc.

ATOM


Atom est un outil d’éditeur de code apprécié par les programmeurs en raison de son interface simple par rapport aux autres éditeurs. Les utilisateurs Atom peuvent soumettre des packages et les utiliser pour le logiciel. Atom est un éditeur de texte avec lequel j’ai déjà travaillé. Je peux le recommander à tout le monde, surtout si vous ne faites que commencer l’apprentissage du code. Atom est développé par GitHub et reste assez jeune dans le domaine des éditeurs de texte.

KOMODO

Komodo Edit est un autre éditeur de texte gratuit avec un design super clair et minimal qui parvient encore à intégrer des fonctionnalités impressionnantes. Différents modes d’affichage sont inclus pour vous permettre d’ouvrir ou de fermer rapidement des fenêtres spécifiques. Le premier est le «mode de mise au point» pour masquer toutes les fenêtres ouvertes et simplement afficher l’éditeur. Les autres affichent / cachent des éléments tels que les dossiers, les résultats du vérificateur de syntaxe et les notifications. Ce programme facilite l’ouverture de documents texte même s’il est actuellement ouvert. Tout en haut du programme se trouve le chemin d’accès au fichier actuellement ouvert, et vous pouvez sélectionner la flèche à côté de n’importe quel dossier pour obtenir une liste de fichiers, qui s’ouvrira comme nouvel onglet dans Komodo Edit si vous le sélectionnez.

SUBLIME TEXT


Sublime Text est un éditeur de texte doté d’un tas de fonctionnalités prenant en charge Windows, Mac OS et Linux. Il est distribué sous forme de logiciel d’évaluation, c’est-à-dire que vous pouvez l’essayer gratuitement sans limite de temps. Vous devrez peut-être payer 70 $ pour obtenir la licence complète. Une autre chose intéressante est que Sublime Text reste rapide et réactif même sur des appareils moins puissants. Cet éditeur de texte dispose de puissantes fonctionnalités de recherche dans l’application. Il existe un mode sans distraction pour vous aider à vous concentrer, en cachant tout le fouillis. Vous pouvez sélectionner plusieurs lignes et les modifier simultanément dans l’éditeur de texte.

NOTEPAD ++


Notepad ++ est probablement l’éditeur de texte libre le plus populaire pour les programmeurs sur Windows. Au début, cela semble être un bloc-notes très simple; Cependant, il contient de nombreuses fonctions intéressantes, qui peuvent être étendues en installant des ajouts. De manière similaire à Sublime Text, Notepad++ possède une grande quantité de modules graphiques supplémentaires. L’application est écrite en C ++ et à cause de cela, elle fonctionne très efficacement (la taille du programme étant très petite). Le programme prend en charge les langues les plus connus connus comme: C, C ++, Java, C #, XML, HTML, PHP, JavaScript, fichier de ressources RC, makefile, ASCII, doxygen, INI pliki, fichiers BATCH, ASP, fichiers source VB / VBS, SQL, Objective -C, CSS, Pascal, Perl, Python, Lua, TCL, Assembleur, Ruby, Lisp, Schéma, Propriétés, Diff, Smalltalk, Postscript i VHDL.
